例如`const char * const * pp`
## 核心口诀：
从右往左:
* const 在 * 的左边：修饰的是指针指向的内容（内容不可变）。
* const 在 * 的右边：修饰的是指针本身（指针的指向不可变）。
## 结论
pp 是一个指针，它指向一个 const 指针，而那个 const 指针又指向一个 const char

------------------------------
## 常见应用场景：字符串数组（只读参数传递）
这种复杂的类型在日常业务中并不是炫技，它有非常重要的工业应用场景——传递一个内容和结构都绝不允许被修改的“只读字符串数组”。
例如，在设计系统的核心 API 时：
```cpp
// 打印系统日志列表。// 我们既不希望函数内部修改字符串内容（**pp），也不希望函数内部把数组里的指针换掉（*pp）
void printSystemLogs(const char * const * pp, int size) {
    for (int i = 0; i < size; ++i) {
        // pp[i] 的底层本质就是 *(pp + i)
        std::cout << pp[i] << std::endl; //  只读访问，绝对安全
    }
}
```
