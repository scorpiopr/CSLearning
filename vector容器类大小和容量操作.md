在 C++ 中，std::vector 的初始化指定元素个数、reserve() 和 resize() 是控制大小（Size）、容量（Capacity）的常用方法。
## 📊 区别对比表

| 特性 / 操作 | vector<T> v(n) (初始化指定个数) | v.reserve(n) (预留空间) | v.resize(n) (重新调整大小) |
|---|---|---|---|
| 大小 size() | 变为 n | 保持不变 | 变为 n |
| 容量 capacity() | 至少为 n | 至少为 n | 至少为 n |
| 是否构造新对象 | 是，立即构造 n 个元素并赋默认值 | 否，只分配原始内存，不构造对象 | 是（若 n > size()），构造新增的元素 |
| 元素析构（减小时） | — (初始化时无法减小) | 否，容量不会因为 reserve 变小 | 是（若 n < size()），销毁超出部分的元素 |
| 安全访问 v[0] 到 v[n-1] | 合法 | ❌ 非法（无对象，属于未定义行为，会崩溃） | 合法 |
| 尾部插入 push_back(x) | 插入到第 n 个位置，大小变为 n+1 | 插入到第 0 个位置，大小变为 1 | 插入到第 n 个位置，大小变为 n+1 |
| 核心使用场景 | 当做固定大小的数组使用，通过下标进行读写。 | 提前规划内存，避免后续 push_back 频繁扩容搬家。 | 动态改变数组大小，截断数据或填充默认数据。 |
