## 指针减法
c++只允许相同类型的指针相减，结果表示两个指针所致内存地址之间所指类型元素内存的块数。
例：
```cpp
#include <iostream>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int* p_start = &arr[0]; // 假设地址为 0x1000
    int* p_end   = &arr[4]; // 地址则为 0x1010 (因为每个 int 占 4 字节)

    // 正确写法：同类型指针相减
    auto diff = p_end - p_start; 

    std::cout << "相减结果: " << diff << std::endl;
    return 0;
}
```
输出为：
```
4
```
### 指针相减返回值的类型
* 是标准库定义的一个特殊有符号整数类型：std::ptrdiff_t（在 <cstddef> 中）。
* 它的字节大小与当前系统的指针大小完全一致（64位系统下是 8 字节，32位系统下是 4 字节）。
* 它是有符号的，如果用前面元素的地址减去后面元素的地址，结果会是负数（如 p_start - p_end 结果为 -4）。
